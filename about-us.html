Background of CodeVerse

The reason I chose to do Assignment 01 on CodeVerse, a website surrounding programming and computers, is that it fits my hobbies and interests. It also allowed me to use a variety of my favourite fonts, in particular American Typewriter and Consolas. The latter, Consolas, is particularly interesting because it's a monospaced font. The following is a brief explanation of what a monospaced font is, and is styled in Consolas-

A monospaced font, also called a fixed-pitch, fixed-width, or non-proportional font, is a font whose letters and characters each occupy the same amount of horizontal space. This contrasts with variable-width fonts, where the letters and spacings have different widths.

Wikipedia

CodeVerse's Programming Languages

View the programming languages supported by CodeVerse below, where you can filter them according to their generally-accepted difficulty to learn. Each programming language has been stored as a JSON object, which allows for easy manipulation and retrieval.

Additionally, feel free to hover over any of the programming language cards to scale it up and change the background colour to white, thereby making it much easier to read.
